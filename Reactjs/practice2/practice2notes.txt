useState = "remember this value, and redraw when it changes"
useEffect = "after redrawing, do this outside thing"
useRef — store a value + changing it does nothing to the screen (and as a bonus, can hold a real DOM element)

useState  →  change value  →  React redraws  ✅
useRef    →  change value  →  React does nothing  🔇
